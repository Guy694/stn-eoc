# Flood Maps Directory

โฟลเดอร์นี้ใช้สำหรับเก็บภาพแผนที่น้ำท่วมรายวันที่บันทึกจากระบบ

## รูปแบบชื่อไฟล์
```
flood-map-YYYY-MM-DD.png
```

ตัวอย่าง:
- `flood-map-2025-12-01.png`
- `flood-map-2025-12-09.png`

## การเข้าถึง
ภาพสามารถเข้าถึงได้ผ่าน URL:
```
http://localhost:3000/flood-maps/flood-map-2025-12-09.png
```

## การจัดการ
- ไฟล์เก่าควรถูกย้ายไปจัดเก็บที่อื่นเป็นระยะ
- แนะนำให้เก็บข้อมูลย้อนหลัง 1 ปี
- สามารถบีบอัดไฟล์เพื่อประหยัดพื้นที่
