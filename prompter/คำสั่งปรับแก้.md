## 2. ความต้องการ (Requirements)

สร้าง **เมนู "วิเคราะห์ (Analytics)"** ที่:
1. **ดึงข้อมูลจาก Database + API ที่มี** มาแสดงผลแบบวิเคราะห์
2. **วิเคราะห์และให้คำแนะนำอัตโนมัติ** เช่น:
   - ควรทำอะไรต่อในสถานการณ์ปัจจุบัน
   - แสดง **จุดอ่อนของแต่ละกลุ่มภารกิจ (ทีม EOC)** ว่า team ไหนขาดแคลน/ทำงานช้า
   - **แจ้งเตือนเวชภัณฑ์เหลือน้อย** (low stock alert) จาก `medical_inventory_stock.balance_qty`
3. **เพิ่ม Chat AI** สำหรับถามคำถามอื่นเพิ่มเติม (ต่อยอดจาก `AIChatbot.jsx`)

- **แท็บโรคระบาด**: แนวโน้มผู้ป่วยรายอำเภอ
- **แท็บ AI วิเคราะห์**: กล่องถามตอบ (Chat AI)

-- ตัวอย่าง: เวชภัณฑ์เหลือน้อย (< 20% ของ opening)
SELECT agency_name, item_name, opening_qty, balance_qty,
       (balance_qty / NULLIF(opening_qty,0)) * 100 AS remain_pct
FROM medical_inventory_stock
WHERE balance_qty / NULLIF(opening_qty,0) < 0.2
ORDER BY remain_pct ASC;
