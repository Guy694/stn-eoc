จงสร้างหน้า ลงทะเบียน สำหรับเจ้าหน้าทีและประชาชน
ให้เลือกลงเป็นประชาชนหรือเจ้าหน้าที่
โดยให้ระบุชื่อ นามสกุล หน่วยงาน ให้ดึงจาก table หน่วย ใน database
ีสำหรับเเจ้าหน้าที่ ให้สามรถเพิ่ม username password แต่ของประชาชนให้เข้าด้วย thaid เท่านั้น
เจ้าหน้าที่เข้าด้วย thaid ได้และ วิธีการเข้า thaid จะเช็คชื่อจาก การลงทะเบียน โดยเช็ค ชื่อ นามสกุล ต้องตรงกัน ถ้าไม่ตรงกัน ให้แจ้งว่าไม่มีข้อมูลผู้ใช้งาน โดยให้อ่านข้อมูลจาก คู่มือthaid.pdf ว่าใช้ parameter ตัวไหน โดยฝั่ง thaid จะใช้ parameter คือ
pid = บัตรประชาชนต้องเข้ารหัส
given_name = ชื่อ
family_name = นามสกุล

ขั้นตอนการป้องกันการกรอกข้อมูลมั่ว
เช็ค validate
ห้ามลงทะเบียนรัวๆใน ip เดียวกัน และอื่นๆ นายเสนอมาได้เลย
