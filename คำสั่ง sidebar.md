Prompt สำหรับจัดกลุ่มเมนู Sidebar ใหม่
คุณคือ Senior UX/UI Designer และ Information Architect สำหรับระบบศูนย์ปฏิบัติการฉุกเฉิน EOC จังหวัด

จากโครงสร้าง Sidebar เดิมของระบบ มีเมนูดังนี้:
- ภาพรวมระบบ
- สภาพอากาศ / Weather Watch
- กลุ่มเปราะบาง
- รายงานเหตุการณ์
- กิจกรรมล่าสุด
- จัดการเจ้าหน้าที่
- จัดการผู้ใช้ประชาชน
- จัดการหน่วยบริการ
- จัดการข้อมูลหมู่บ้าน
- จัดการ EOC
- ประวัติ EOC Sessions
- บุคลากร
- ยานพาหนะ
- อุปกรณ์
- ศูนย์พักพิง
- ทรัพยากร IT

ต้องการจัดกลุ่มเมนู Sidebar ใหม่ให้ใช้งานง่ายขึ้น เป็นระบบมากขึ้น และรองรับเมนูเพิ่มเติม ได้แก่:
- เวชภัณฑ์และคงคลัง: ใช้สำหรับบันทึกจำนวนยา เวชภัณฑ์ วัสดุการแพทย์ และรายการคงคลังอื่น ๆ เพื่อใช้ในการเบิก จ่าย ติดตาม และบริหารทรัพยากรเมื่อเปิด EOC

เป้าหมายการออกแบบ:
1. จัดกลุ่มเมนูใหม่ให้เป็นหมวดหมู่ที่เข้าใจง่าย
2. ลดความซ้ำซ้อนระหว่าง “จัดการเจ้าหน้าที่” กับ “บุคลากร”
3. แยกเมนูสำหรับผู้บริหาร เจ้าหน้าที่ปฏิบัติการ และผู้ดูแลระบบให้ชัดเจน
4. รองรับการขยายเมนูในอนาคต เช่น อาหาร น้ำดื่ม เครื่องมือสื่อสาร น้ำมันเชื้อเพลิง หรือระบบคลังอื่น ๆ
5. ใช้ชื่อเมนูภาษาไทยที่เป็นทางการ แต่เข้าใจง่าย
6. ใช้ icon ที่เหมาะสมจาก Lucide React
7. โครงสร้างต้องเหมาะกับระบบ EOC, ระบบราชการ, ระบบสาธารณสุข และ Dashboard เชิงปฏิบัติการ

ให้จัดกลุ่มเมนูใหม่เป็นโครงสร้าง Sidebar โดยแบ่งเป็นหมวดหลักและเมนูย่อย พร้อมอธิบายเหตุผลของการจัดกลุ่มแต่ละหมวด

ข้อกำหนดเพิ่มเติม:
- เมนู “เวชภัณฑ์และคงคลัง” ควรอยู่ในกลุ่ม “ทรัพยากรและโลจิสติกส์”
- เมนู “บุคลากร”, “ยานพาหนะ”, “อุปกรณ์”, “ศูนย์พักพิง”, “ทรัพยากร IT” ควรอยู่ในกลุ่มเดียวกัน
- เมนู “จัดการเจ้าหน้าที่” ควรอยู่ในกลุ่ม “ผู้ใช้งานและสิทธิ์”
- เมนู “จัดการผู้ใช้ประชาชน” ควรอยู่ในกลุ่ม “ข้อมูลประชาชน”
- เมนู “จัดการ EOC” และ “ประวัติ EOC Sessions” ควรอยู่ในกลุ่ม “บริหาร EOC”
- เมนู “รายงานเหตุการณ์” และ “กิจกรรมล่าสุด” ควรอยู่ในกลุ่ม “ปฏิบัติการเหตุการณ์”
- เมนู “สภาพอากาศ / Weather Watch” ควรอยู่ในกลุ่ม “เฝ้าระวังสถานการณ์”

ผลลัพธ์ที่ต้องการ:
1. เสนอ Sidebar menu structure ใหม่
2. ระบุชื่อหมวดหลัก
3. ระบุเมนูย่อย
4. ระบุ icon ที่เหมาะสม
5. ระบุ route/path ตัวอย่าง
6. เสนอการจัดลำดับเมนูจากบนลงล่าง
7. เสนอ UX improvement เพิ่มเติม เช่น badge, count, alert indicator, collapse group, active state
